# Security Policy

## Supported Versions

It is always going the be the latest version having support for security updates. Any older and outdated versions are not going to have security updates, being either major or minor.

## Reporting a Vulnerability

To report a vulnerability, you can either contact me by email ([j.pedro.ferreira@ipvc.pt](mailto:j.pedro.ferreira@ipvc.pt)) or post the security vulnerability in this section.
